Symbol Table

Features:

Currently Operational:

Type Definitions - Capable of handling simple variable types(long, char, int, etc.).
Arrays - Fully Functional with symbol table.
Pointers - Fully Functional with symbol table.
Functions - Fully Functional with symbol table.
Constants - Fully Functional with symbol table.
Recognize Syntax Errors - Can locate errors and provide feedback.
Looping - Fully Functional with symbol table.
Constant Value Arithmetic - Capable of optimizing simple arithmetic.

Dependencies:

Python - Main Language of this Compiler Project
PiP - This module allows for easy of use in installing these python modules.
PLY - This module mimics the C equivalent of Flex and Yacc
Bintrees - This module allows for an easy of use Red-Black Tree
sudo apt-get install python-pip
sudo pip install ply bintrees

Instructions:

To run the driver for the symbol table, please use this command.
python symboltable_driver.py
