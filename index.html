CS660 Compiler Construction

Graduate Team: Chase Carthen, Nolan Burfield, Vinh Le

Overview

Basic Info:

This website is to serve as a download hub for a project team in
Dr. Harris' CS660 Compiler Construction course. The goal of this
project is to create a fully functional compiler capable of converting
from ANSI C to MIPS. This project is written completely in the Python
programming language, with the help of Python Modules: PLY, Bintrees,
and various others. Various changes will be made available as time goes
on. We would like to mention that we took influence from Pycparser
in the creation of our AST Visitor Class. Thank you and enjoy!

Implementations:

Currently Operational:

Symbol Table - Possesses a Function Node, an Array Node, a Variable Node, and Pointer Node. Currently utilizes a Red-Black Tree.
Scanner - Capable of handling Lexical Errors. Errors are not terminating the program due to testing purposes.
Instead of !!s, it has been opted for @.
Parser - Parser serves as the main hub of communication between the Scanner, AST, and 3AC generator.
AST - AST is auto generated via a generator script file and a Visitor class was made to traverse the tree. 3AC - This utilized the Visitor class and traversed the tree to generated the appropriate 3AC code.

Assumptions Made:

Please don't be smartass:

You are not inputing headerfiles.
You are not inputing obscure unsigned or long types.
You are not Terence.
The only supported types right now are Ints, Chars, Floats.
Only 1D pointers
No Array of Structs

Restrictions:

Current Notes:

The project itself only fails on grammar errors.
This is because we desire to see entire compile run through for testing purposes.

Directions

Dependencies:

Python - This project was developed in the Python programming language.
PiP - We used pip in order to install these python modules.
PLY - We used this python equivalent of Flex and Yacc in order to deal with the input character stream.
BinTrees - We used BinTrees in order to insert balanced Binary Trees within into the symbol table.
StrConv - This module allows for the type inference needed for type checking
TermColor - Chase has OCD for coloring text
sudo apt-get install python-pip
sudo pip install ply bintrees strconv termcolor

Command Line Arguments:

-p : Scan File
-h : Help Message
-v : Version Info
-t : Token Log Output

How to Run:

Click on any of the projects to see a preview of the code or download the entirety at the end
of the section.
To run this project, merely run the script from command line.
./demo1.sh
To run the driver for the symbol table, please use this command.
python symboltable_driver.py
